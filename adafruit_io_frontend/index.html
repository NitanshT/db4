Remote controls

These controls write to Adafruit IO feeds. The ESP32 subscribes to these feeds and updates the relay/Peltier logic remotely.

setpoint-temp peltier-enable auto-control manual-peltier test-number test-duration-s

Remote writes require the AIO key. Do not hard-code it in the repository; enter it at runtime.
